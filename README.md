# pipeline-tools

Home for the Dockerfile of the deniceg/pipeline-tools

It contains:
* kubectl
* kubecrypt
* kubeinfo
* yaml-renderer
* sensuctl
* bash
* curl
* git
* docker
* make
* jq
* bind-tools

To update something, adjust the versions stated in the Dockerfile.

## Use it example
```
docker pull deniceg/pipeline-tools:v1.0.0
```
